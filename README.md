# Hello World App

A simple Hello World web app built with Python and Flask.

## Features

- Click a button to display "Hello!"
- Flask backend with a `/hello` API endpoint
- Clean, minimal UI

## Requirements

- Python 3
- Flask

## Installation

```bash
pip3 install flask
```

## Usage

```bash
python3 app.py
```

Then open [http://127.0.0.1:5000](http://127.0.0.1:5000) in your browser and click **Say Hello**.
